## Project Name:KeenKeeper
## Description:
An elegant, data-driven personal application engineered to track, manage, and optimize communication within social circles.

## Tech Stack Architecture
- React.js 
- React Router DOM 
- Tailwind CSS Framework

## Key Features:
1) Dynamically transforms user cards. 
2) Real-time interaction streaming logs.
3) Generates a composite interactive modular Pie Chart visualization.
